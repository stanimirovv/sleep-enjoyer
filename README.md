# 🌙 Baby Sleep - Brownian Noise Generator

A minimalist web app that generates soothing Brownian noise to help babies sleep. Designed for dark rooms with a mobile-optimized interface.

## Features

- **True Brownian Noise**: Warmer, bass-heavy sound (not harsh white noise)
- **Dark Mode**: Deep navy background (#0f172a) to protect eyes in dark rooms
- **Mobile Optimized**: Large, easily tappable button with responsive design
- **Continuous Playback**: Seamless audio generation with no hiccups
- **Screen Wake Prevention**: Attempts to keep mobile screens awake during playback
- **Zero Dependencies**: Pure vanilla JavaScript, HTML, and CSS
